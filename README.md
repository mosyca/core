## About

Mosyca Core is a PHP/Symfony framework for building MCP-native, self-documenting API plugins. Write once, expose everywhere — REST, CLI, MCP tool, OpenAPI spec — with zero boilerplate.

## Installation

```bash
composer require mosyca/core
```
